Commencer une session
=====================
- Aller sur l'URL en .../emotions2 pour accéder au formulaire de consentement + infos détaillés sur le juge/sujet

- Aller sur .../emotions/miniform.html pour accéder au formulaire minimum avant la session


Fichiers son
============

Le répertoire 'sounds' contient les fichiers sonores à utiliser comme phrases.
Un tri alphabétique sur les fichiers est effectué pour déterminer l'ordre de passage. Il est donc conseillé de préfixer les noms de fichier par leur numéro sur deux digits (ex: 01.phrase1.wav).

Les fichiers doivent être au format .wav

Fichiers de sortie
==================

 Ils sont générés dans le répertoire 'out' avec le prénom du juge/sujet.
